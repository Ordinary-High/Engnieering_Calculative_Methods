## 常用Excel函数
在 WPS Excel 中，这四个函数是进行矩阵运算和数组操作的有力工具。前三个 `MMULT`, `MINVERSE`, `TRANSPOSE` 是经典的数学和三角函数，而 `HSTACK` 是 WPS 引入的新一代动态数组函数，二者结合能极大地提升数据处理效率。

### 📈 MMULT：矩阵乘法函数

**MMULT** 函数用于计算两个数组的矩阵乘积。

*   **语法**：`=MMULT(array1, array2)`
*   **参数说明**：
    *   `array1` (必需)：进行乘法运算的第一个数组。
    *   `array2` (必需)：进行乘法运算的第二个数组。
*   **计算规则**：结果的**行数**与 `array1` 相同，**列数**与 `array2` 相同。
*   **使用示例**：
    *   假设有一个 2x3 的矩阵 A (A1:C2) 和一个 3x2 的矩阵 B (E1:F3)。
    *   选中一个 2x2 的空白区域 (如 H1:I2) 作为输出区域。
    *   在编辑栏输入公式：`=MMULT(A1:C2, E1:F3)`
    *   按下数组公式组合键 **`Ctrl + Shift + Enter`** 确认。
*   **注意事项与限制**：
    *   `array1` 的列数必须等于 `array2` 的行数，否则返回 `#VALUE!` 错误。

### 🔢 MINVERSE：矩阵求逆函数

**MINVERSE** 函数用于计算一个方阵的逆矩阵。

*   **语法**：`=MINVERSE(array)`
*   **参数说明**：
    *   `array` (必需)：要计算逆矩阵的数组，必须是行数和列数相等的**方阵**。
*   **使用示例**：
    *   假设有一个 2x2 的方阵 A (A1:B2)。
    *   选中一个同样为 2x2 的空白区域 (如 D1:E2) 作为输出区域。
    *   在编辑栏输入公式：`=MINVERSE(A1:B2)`
    *   按下数组公式组合键 **`Ctrl + Shift + Enter`** 确认。
*   **注意事项与限制**：
    *   `array` 中的单元格必须全部为数值，空白或文本会导致 `#VALUE!` 错误。
    *   如果矩阵的行列数不相等，同样返回 `#VALUE!` 错误。
    *   结果是一个数组，必须以 **数组公式** 形式输入。

### 🔄 TRANSPOSE：矩阵转置函数

**TRANSPOSE** 函数可以将一个数组或区域的行和列进行互换。

*   **语法**：`=TRANSPOSE(array)`
*   **参数说明**：
    *   `array` (必需)：需要进行转置的数组或单元格区域。
*   **使用示例**：
    *   假设有一个 2 行 3 列的区域 A1:C2 需要转置。
    *   选中一个 3 行 2 列的空白区域 (如 E1:F3) 作为输出区域。
    *   在编辑栏输入公式：`=TRANSPOSE(A1:C2)`
    *   按下数组公式组合键 **`Ctrl + Shift + Enter`** 确认。


### 📦 HSTACK：水平拼接函数

**HSTACK** 是 WPS 引入的新一代动态数组函数，它可以在水平方向上将多个数组或区域拼接成一个新数组。

*   **语法**：`=HSTACK(array1, [array2], ...)`
*   **参数说明**：
    *   `array1`：要拼接的第一个数组，这是必需的。
    *   `array2, ...`：要拼接的后续数组，均为可选。
*   **限制**：

    *   **行数必须一致**：若要水平拼接，所有参与拼接的数组必须具有**相同的行数**，否则空缺的位置将显示 `#N/A` 错误。
*   **实用技巧**：`HSTACK` 常与它的“兄弟”函数 `VSTACK` (垂直拼接) 配合使用，实现复杂的数据重组。


### 🎯 SUMPRODUCT：向量点乘 / 多条件求和函数

**SUMPRODUCT** 是 Excel / WPS 中一个非常强大且多功能的函数。它的核心功能是将多个数组中对应位置的元素相乘，然后对乘积结果求和，这正好就是数学中的**向量点乘（内积）**。同时，它还支持条件判断，可以实现多条件计数、多条件求和等高级计算，且无需按数组公式组合键（`Ctrl+Shift+Enter`），直接回车即可。

*   **语法**：`=SUMPRODUCT(array1, [array2], [array3], ...)`

*   **参数说明**：
    *   `array1` (必需)：第一个数组或区域，其元素将被参与相乘并求和。
    *   `[array2], [array3], ...` (可选)：最多可提供 255 个数组。这些数组必须具有相同的维度（即行数和列数一致），否则返回错误值 `#VALUE!`。

*   **注意事项与限制**：
    *   **维度必须相同**：所有数组参数的**行数**和**列数**必须完全一致，否则返回 `#VALUE!`。

### 📈 TTEST：T检验函数

**TTEST** 函数用于返回与学生 t 检验相关的概率（即 **p 值**），以此判断两个样本是否可能来自具有相同平均值（即无显著差异）的总体。该函数的核心作用是帮助我们区分两组数据的差异是“偶然波动”还是“真实存在”。

*   **语法**：`=TTEST(array1, array2, tails, type)`

*   **参数说明**：
    *   `array1` (必需)：第一个数据集，须为单元格区域或数组。
    *   `array2` (必需)：第二个数据集，其数据点个数须与 `array1` 相同，否则在某些类型下会返回错误。
    *   `tails` (必需)：指示分布曲线的尾数，只能为 1 或 2。
        *   `tails = 1`：**单尾检验**，用于判断一组数据是否显著**大于**或**小于**另一组，针对有明确方向的假设（如“改进后的产品销量更高”）。
        *   `tails = 2`：**双尾检验**，仅判断两组数据**是否存在差异**，而不关心方向，在日常工作中最为常用。
    *   `type` (必需)：t 检验的类型，共有三种选择：
        *   `type = 1`：**配对检验**，适用于同一组样本在前后两个时间点的对比（如用户使用产品前后的满意度评分）。
        *   `type = 2`：**等方差检验**（双样本等方差假设），适用于两组独立数据且它们的波动幅度（方差）相近（如两个部门的绩效考核）。
        *   `type = 3`：**异方差检验**（双样本异方差假设），适用于两组独立数据且它们的波动幅度差异较大，当无法确定方差是否相等时，使用此类型更为稳妥。

*   **使用示例**：
    假设你想对比新产品与老产品的销量是否存在显著差异，销量数据分别位于 `A2:A6` 和 `B2:B6`。
    1.  选中任意空白单元格（如 `C1`）。
    2.  输入公式：`=TTEST(A2:A6, B2:B6, 2, 3)`
    3.  直接按 `Enter` 键确认，结果会返回一个介于 0 和 1 之间的概率值（p 值）。
        *   若 p 值 **< 0.05**，通常认为差异是显著的，新产品销量显著不同于老产品。
        *   若 p 值 **≥ 0.05**，则认为差异可能是由偶然因素导致的。

*   **注意事项与限制**：
    *   `array1` 和 `array2` 必须为数值，任何空白、逻辑值或文本单元格都将被视为无效数据。
    *   当 `type = 1`（配对检验）时，`array1` 与 `array2` 的数据点个数必须相同，否则返回错误值 `#N/A`；其他类型下个数不同则函数会自动处理。

### 📏 TINV：T分布分位数计算函数

**TINV** 函数用于返回作为给定概率（双尾）和自由度函数的学生 t 分布的 **t 值**（即临界值）。该函数的典型应用场景是根据置信水平计算置信区间，或进行假设检验中的临界值确定。

*   **语法**：`=TINV(probability, degrees_freedom)`

*   **参数说明**：
    *   `probability` (必需)：对应于双尾学生 t 分布的概率，其值必须在 `(0, 1]` 区间内。
        *   在实际应用中，它通常等于 **1 - 置信水平**。例如，计算 95% 置信区间时，此处输入 `0.05`。
    *   `degrees_freedom` (必需)：分布的自由度。
        *   在单样本场景中，自由度 = **样本数量 - 1**，且必须为正整数。

*   **使用示例**：
    假设有 10 个样本，需要计算 95% 置信水平下对应的 t 分布临界值。
    1.  选中任意空白单元格（如 `D1`）。
    2.  输入公式：`=TINV(0.05, 9)` （因为 1 - 0.95 = 0.05，自由度 = 10 - 1 = 9）。
    3.  直接按 `Enter` 键确认，结果约为 `2.2622`。这个值是在进行 t 检验时判断统计量是否显著的临界点。

*   **注意事项与限制**：
    *   `probability` 必须介于 0 和 1 之间（可以等于 1，但不能等于 0），否则返回 `#NUM!` 错误。
    *   `degrees_freedom` 必须 ≥ 1，若 `< 1` 则返回 `#NUM!` 错误；若非整数，则函数会将其截尾取整。
    *   若任一参数为非数值型，函数返回错误值 `#VALUE!`。


针对您这次提出的两个函数，以下是按照我们约定的格式进行的详细介绍。它们分别用于衡量一组数据的“中心”和“离散”程度，是统计学中最核心的两个基础指标。

### 📊 AVERAGE：均值计算函数

**AVERAGE** 函数返回一组数值的算术平均数（即总和除以个数），是日常办公中使用频率最高的统计函数之一。

*   **语法**：`=AVERAGE(number1, [number2], ...)`

*   **参数说明**：
    *   `number1`（必需）：需要计算平均值的第一个数值、单元格引用或区域。
    *   `[number2], ...`（可选）：最多可包含 255 个附加数值或区域。

*   **使用示例**：
    *   计算A1至A10单元格中销售数据的月均销售额：
        公式为 `=AVERAGE(A1:A10)`，按下回车键即可得到结果。
    *   计算不连续区域（B2:B7和D2:D7）的销售额平均值：
        公式为 `=AVERAGE(B2:B7, D2:D7)`。

*   **注意事项**：
    *   参数可以是数字，或者是包含数字的名称、数组或引用。
    *   如果数组或引用参数包含文本、逻辑值或空白单元格，这些值将被自动忽略；但包含**零值**的单元格会计算在内。


### 📏 STDEV：样本标准差计算函数

**STDEV** 函数用于估算基于**样本**的标准偏差。它反映了数值相对于其平均值 (Mean) 的离散程度，是衡量数据波动性的关键指标。

*   **语法**：`=STDEV(number1, [number2], ...)`

*   **参数说明**：
    *   `number1`（必需）：对应于总体样本的第一个数值参数。
    *   `[number2], ...`（可选）：最多可包含 255 个附加数值参数。

*   **使用示例**：
    *   假设A1至A10单元格包含了某产品的抽样检测数据，要估算这批样本数据的标准偏差：
        公式为 `=STDEV(A1:A10)`，按下回车键即可得到结果。

*   **注意事项**：
    *   函数 `STDEV` 假设其参数是总体中的**样本**，使用的是“无偏差”或“n-1”方法。
    *   如果数据代表**全部样本总体**（即完整数据集），则应该使用 `STDEVP` 或 `STDEV.P` 函数来计算标准偏差。
    *   `STDEV` 是旧版函数，在 WPS 中依然保留和使用。推荐优先使用新版本函数 `STDEV.S`，其含义相同但命名更清晰。
