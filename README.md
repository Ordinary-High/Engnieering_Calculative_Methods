## 仓库介绍
该仓库创建初衷是为了作者自己复习和整理课程代码以便考试使用，但受开源思想的影响，决定分享出来供复习参考
### 1. PPT来源
2025-2026学年春学期，刘杰老师上课用的PPT版本
### 2. 仓库结构
- [Code](Code/)文件夹下存放了参考代码，为了在考试考试时可以快速复用，作者大量采用宏定义和全局变量，使用时复制粘贴并且参照注释在对应的地方修改即可。
- [PPT](PPT/)文件夹下存放了上课所用的PPT。
- [Summary](Summary/)文件夹下存放了作者的知识总结，按照知识点归为文件夹，部分知识的内容讲了两节课也归为同一个文件夹下，**建议复习时从这个文件夹开始**，文档已经在合适的连接到了对应的文件。
- [Excel](Excel/)文件夹下存放了所使用的Excel文件，并总结了常用的函数。
- [计算方法（合并）(2).pdf](计算方法（合并）(2).pdf)为电子版课本。
